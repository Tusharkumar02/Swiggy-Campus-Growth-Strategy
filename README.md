# StrategEats | Swiggy Campus Food-Tech Business & Growth Strategy

This project focuses on analyzing campus food consumption behavior and designing
a growth strategy to position Swiggy as the default food companion for students.

## Key Highlights
- Consumer behavior analysis of campus food usage
- Market sizing and revenue estimation (GMV-focused)
- Growth and retention strategies using loyalty and engagement mechanisms
- Structured strategy presentation summarizing insights and recommendations

## Tools Used
- Microsoft Excel
- Microsoft PowerPoint
- Secondary Research

## Deliverable
- Strategy presentation deck outlining analysis and recommendations
